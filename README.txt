Palindrome is a word, phrase, number, or other sequence of characters which reads the same backward or forward. Allowances may be made for adjustments to capital letters, punctuation, and word dividers.

This is my very basic palindrome written in Node.js. 

Step 1. Install node via brew:
a. Open 'Terminal' 
b. Enter command, 'brew install node'
c. Verify install, 'node --version'

Step 2. Download node and install:
a. Vist http://nodejs.org/download/
b. Download the correct install for your system
c. Open 'Terminal' and verify install, 'node --version'

Steps to run:
1. Open 'Terminal'
2. Enter command, 'node [PATH_TO_PALINDOME_CODE]'
3. To exit script, enter 'exit' or Control + C
